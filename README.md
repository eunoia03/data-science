# Data Science

This repository documents my journey learning data science, machine learning, and data analytics.

The goal of this repository is to build a strong understanding of core concepts while applying them through practical mini-projects. Rather than only collecting finished projects, I also maintain concise examples and notes that I can revisit as a personal reference.
